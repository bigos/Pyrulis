* Graph Viewer
A simple utility to view cons trees and other data structures. This is still
work in progress.

#+begin_src lisp
  ;; example use
  (push #p"~/Programming/Pyrulis/Lisp/graph-viewer/" asdf:*central-registry*)
  (ql:quickload :graph-viewer)
  (graph-viewer:graph '(1 (2.1 . 2.2) 3))
#+end_src
